# AutoShop Filter Widget

Vue 3 + TypeScript project scaffolded with Vite and structured for future embeddable widget/library builds.

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Preview

```bash
pnpm preview
```
